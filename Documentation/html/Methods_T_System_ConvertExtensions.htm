ConvertExtensions 형식은 다음의 멤버를 노출합니다.

메서드
  이름설명
ToT
value를 특정 형식으로 변환하고, value가 null이거나 변환 중 예외 발생 시 기본 값을 반환.
ToNullableT
value를 특정 형식에 대한 Nullable로 변환하고, value가 null이거나 변환 중 예외 발생 시 null을 반환.
TryConvertT
value를 특정 형식으로 변환 시도
