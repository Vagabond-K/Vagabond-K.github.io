    VagabondK.Indicators 네임스페이스

    데이터를 화면에 표시하기 위한 인디케이터 관련 라이브러리입니다.

    클래스

    DelegateDrawing

    대리자를 이용한 파트 드로잉을 정의합니다.

    DigitalIndicatorExporter

    디지털 인디케이터 내보내기 기능을 제공합니다.

    DigitalNumberExtentions

    IDigitalNumber 인터페이스에 대한 확장 메서드 모음입니다.

    DigitalNumberPresenterExtentions

    IDigitalNumberPresenter 인터페이스에 대한 확장 메서드 모음입니다.

    DigitalTextExtentions

    IDigitalText 인터페이스에 대한 확장 메서드 모음입니다.

    DigitalTextPresenterExtentions

    IDigitalTextPresenter 인터페이스에 대한 확장 메서드 모음입니다.

    ParametricCache<T>

    파라미터 기반 객체 보관 캐시입니다.

    PartDrawing

    파트 드로잉을 정의합니다.

    PartDrawingContextExtensions

    IPartDrawingContext 인터페이스의 확장 메서드 모음입니다.

    PartDrawingContext<TRenderer>

    파트 드로잉 컨텍스트. 파트의 모양을 패스 기반으로 그리기 위한 메서드를 제공합니다.

    PartDrawingContext<TRenderer, TRendererPoint>

    파트 드로잉 컨텍스트. 파트의 모양을 패스 기반으로 그리기 위한 메서드를 제공합니다.

    SvgPathDrawingContext

    파트 드로잉 컨텍스를 통해 SVG 패스 태그 구문을 생성합니다.

    구조체

    Part

    인디케이터를 구성하는 파트를 정의합니다.

    인터페이스

    IDigitalIndicator

    값을 여러 세그먼트로 구성된 디지털 문자들로 표시하는 인디케이터입니다.

    IDigitalIndicatorPresenter

    표시할 세그먼트에 대한 필터에 따라 디지털 문자들을 표시합니다.

    IDigitalNumber

    수치형 값을 디지털 문자들로 표시하는 인디케이터입니다.

    IDigitalNumberPresenter

    세그먼트 상태에 따라 수치형 값을 디지털 문자들을 표시합니다.

    IDigitalText

    값을 문자열 형식으로 변환하여 디지털 문자들로 표시하는 인디케이터입니다.

    IDigitalTextPresenter

    세그먼트 상태에 따라 값을 문자열로 변환하여 디지털 문자들을 표시합니다.

    IIndicator

    값을 화면에 표시하는 인디케이터를 정의합니다.

    IPartDrawingContext

    파트 드로잉 컨텍스트 인터페이스. 파트의 모양을 패스 기반으로 그리기 위한 메서드를 정의합니다.
