HoldingRegisterAttribute 형식은 다음의 멤버를 노출합니다.

Address

(ModbusPointAttribute에서 상속됨.)
BitFlagIndex
Bit 플래그 인덱스. Word 단위 데이터에서의 Bit 위치를 설정함. BitFlagPoint에서 사용.
(WordAttribute에서 상속됨.)
BytesLength
Byte 단위 설정 길이. DateTime, byte[], string 형식에서 사용함.
(WordAttribute에서 상속됨.)
DateTimeFormat
DateTime 값의 직렬화/역직렬화 형식
(WordAttribute에서 상속됨.)
DateTimeFormatString
DateTime 값의 직렬화/역직렬화를 위한 형식 문자열
(WordAttribute에서 상속됨.)
DateTimeKind
Modbus 데이터 상 DateTime 값의 DateTimeKind를 정의
(WordAttribute에서 상속됨.)
Encoding
string 형식 값의 직렬화/역직렬화를 위한 인코딩
(WordAttribute에서 상속됨.)
Endian
Modbus 엔디안
(WordAttribute에서 상속됨.)

(InterfacePointAttribute에서 상속됨.)
Scale
수치형 값의 스케일(배율)
(WordAttribute에서 상속됨.)
SkipFirstByte

SlaveAddress
