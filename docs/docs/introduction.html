    Vagabond K의 .NET 라이브러리

    개인적으로 만든 라이브러리에 대한 설명을 API 레퍼런스 형식으로 제공하고자 본 사이트를 만들었습니다.
    얼마나 유용할지는 모르지만, 혼자 쓰는 것 보다는 필요한 분들과 함께 쓰는 것이 더 좋을 것 같아 게시했습니다.

    주된 사용법은 본 사이트 및 개인 블로그를 참고하시면 되며, 문의 사항이 있으시면 블로그에 댓글을 주시거나 카카오톡 오픈채팅을 통해 문의를 주셔도 좋습니다.

    즐거운 코딩 시간 되시길 바랍니다.

    전체 네임스페이스 목록

    System

    개인적으로 워낙 자주 써서 그냥 System 네임스페이스에 추가해버린 기능들입니다.

    System.ComponentModel

    개인적으로 워낙 자주 써서 그냥 System.ComponentModel 네임스페이스에 추가해버린 기능들입니다.

    System.Threading

    개인적으로 워낙 자주 써서 그냥 System.Threading 네임스페이스에 추가해버린 기능들입니다.

    VagabondK.App

    응용프로그램 개발 관련 지원 라이브러리입니다. 최상위 Shell 및 서비스 정의 등을 포함합니다.

    VagabondK.App.Windows

    Windows 운영체제 기반 응용프로그램 개발 관련 지원 라이브러리입니다. VagabondK.App 어셈블리에 정의된 서비스들의 Windows 운영체제용 구현을 포함합니다.

    VagabondK.Indicators

    데이터를 화면에 표시하기 위한 인디케이터 관련 라이브러리입니다.

    VagabondK.Indicators.Avalonia

    데이터를 화면에 표시하기 위한 Avalonia UI용 인디케이터 관련 라이브러리입니다.

    VagabondK.Indicators.DigitalFonts

    디지털 인디케이터를 위한 디지털 문자 양식들을 제공합니다.

    VagabondK.Indicators.GeometryUtil

    인디케이터를 화면에 그리기 위한 요소 및 기능들을 제공합니다.

    VagabondK.Indicators.Maui

    데이터를 화면에 표시하기 위한 MAUI용 인디케이터 관련 라이브러리입니다.

    VagabondK.Indicators.PartBuilders

    캐시를 통한 파라미터 기반 인디케이터 파트를 관리하는 기능들을 제공합니다.

    VagabondK.Indicators.Razor

    데이터를 화면에 표시하기 위한 Razor 기반 인디케이터 관련 라이브러리입니다.

    VagabondK.Indicators.Windows

    데이터를 화면에 표시하기 위한 Windows용 인디케이터 관련 라이브러리입니다.

    VagabondK.Interface

    프로토콜 기반 통신을 이용한 데이터 인터페이스에 대한 기본 기능을 제공합니다.

    VagabondK.Interface.Abstractions

    프로토콜 기반 통신을 이용한 데이터 인터페이스 기능 구현을 위한 추상 정의를 제공합니다.

    VagabondK.Interface.LSElectric

    LS ELECTRIC(구 LS산전)의 Cnet, FEnet 등의 프로토콜을 이용한 데이터 인터페이스에 대한 기능들을 제공합니다.

    VagabondK.Interface.LSElectric.Abstractions

    LS ELECTRIC(구 LS산전)의 Cnet, FEnet 등의 프로토콜을 이용한 데이터 인터페이스 기능 구현을 위한 추상 정의를 제공합니다.

    VagabondK.Interface.Modbus

    Modbus 프로토콜을 이용한 데이터 인터페이스에 대한 기능들을 제공합니다.

    VagabondK.Interface.Modbus.Abstractions

    Modbus 프로토콜을 이용한 데이터 인터페이스 기능 구현을 위한 추상 정의를 제공합니다.

    VagabondK.Korean.Hangul

    한글과 관련된 기능들을 지원하는 라이브러리입니다.

    VagabondK.Korean.ScreenInput

    한국어의 화면 기반 입력 기능들을 지원하는 라이브러리입니다.

    VagabondK.Protocols

    여러 프로토콜들의 관련 기능을 모아놓은 라이브러리입니다.

    VagabondK.Protocols.Channels

    여러 프로토콜들의 통신 채널 관련 기능들을 제공합니다.

    VagabondK.Protocols.Logging

    여러 프로토콜들의 통신 채널 로깅 관련 기능들을 제공합니다.

    VagabondK.Protocols.LSElectric

    LS ELECTRIC(구 LS산전) 제품 관련 라이브러리에서 기본적으로 사용되는 프로그래밍 요소들이 포함되어 있습니다.

    VagabondK.Protocols.LSElectric.Cnet

    LS ELECTRIC(구 LS산전) Cnet 프로토콜 라이브러리에서 기본적으로 사용되는 프로그래밍 요소들이 포함되어 있습니다.

    VagabondK.Protocols.LSElectric.Cnet.Simulation

    LS ELECTRIC(구 LS산전) Cnet 프로토콜 시뮬레이션 관련 기능들을 제공합니다.

    VagabondK.Protocols.LSElectric.FEnet

    LS ELECTRIC(구 LS산전) FEnet 프로토콜 라이브러리에서 기본적으로 사용되는 프로그래밍 요소들이 포함되어 있습니다.

    VagabondK.Protocols.LSElectric.FEnet.Simulation

    LS ELECTRIC(구 LS산전) Cnet 프로토콜 시뮬레이션 관련 기능들을 제공합니다.

    VagabondK.Protocols.Modbus

    Modbus 라이브러리에서 기본적으로 사용되는 프로그래밍 요소들이 포함되어 있습니다.

    VagabondK.Protocols.Modbus.Data

    Modbus 라이브러리의 데이터 처리 관련 기능들을 제공합니다.

    VagabondK.Protocols.Modbus.Serialization

    Modbus 라이브러리의 통신 프레임 직렬화 관련 기능들을 제공합니다. Modbus RTU, Modbus TCP, Modbus ASCII를 지원합니다.

    VagabondK.Windows

    Windows 운영체제 기반 개발 관련 라이브러리입니다.

    VagabondK.Windows.Commands

    자주 사용되는 커맨드들을 지원합니다.

    VagabondK.Windows.Converters

    MVVM을 위한 컨버터들을 지원합니다.
