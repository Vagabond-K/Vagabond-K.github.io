    DecimalSeparatorSize

    IntegerDigits

    표시할 정수 자리 개수을 가져오거나 설정합니다.
    MinusAlignLeft

    마이너스 기호를 왼쪽 끝에 정렬할지 여부를 가져오거나 설정합니다.
    PadZeroLeft

    왼쪽(정수 자리) 빈칸을 0으로 채울지 여부를 가져오거나 설정합니다.
    PadZeroRight

    오른쪽(소수 자리) 빈칸을 0으로 채울지 여부를 가져오거나 설정합니다.
